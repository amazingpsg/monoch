# 📜 일꾼 행동 수칙 (Project Rules)

나리께서 내리신 지엄하신 훈육 장부이옵니다. 밭을 갈거나 새끼줄을 꼴 때 반드시 이 규칙을 기본으로 삼아 일하겠나이다.

1. **그림 자원은 무조건 분리하기**
   - PDF나 거대한 이미지를 HTML 웹 문서로 변환하는 작업을 시작할 때는, 처음부터 그림 파일들을 `images/` 폴더로 칼같이 분리하여 보관하며, HTML 장부 내에 직접 글자 코드(Base64)로 쑤셔 넣지 않는다.

2. **화면 크기에 맞춰 그림 다이어트하기 (리사이징)**
   - 나리께서 모니터나 폰으로 보실 화면 크기(예: 가로 800px 등)에 알맞게 그림 해상도를 축소(resize)하여 불필요한 픽셀 거품을 걷어낸다.

3. **포맷은 무조건 WebP로 굽기**
   - 저장하는 그림 포맷은 화질은 그대로 살리고 용량 바람은 쫙 빼주는 차세대 고압축 웹 이미지 포맷인 `WebP`로 무조건 인코딩하여 전체 파일 용량을 최소화한다.
